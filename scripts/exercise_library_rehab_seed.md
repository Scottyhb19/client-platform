# Odyssey — Exercise Library Seed (Foundation / Rehab & Mobility)

**Purpose.** The lower-level companion to the loaded library. Early-stage rehab and mobility work — low load, controlled, regression-friendly. This is deliberately the tier the loaded library excludes.

**How to use.** Review and edit directly, then hand to Claude Code. Each line under a heading is one exercise. Format is `Name SETSxREPS METRIC`. Held work (stretches, isometric holds) uses time: `1x45sec`. Rep-based work uses `2x12` with metric `-` (bodyweight), `KG` (light load), or `m` (carry distance). Lines tagged `(machine)` are machine-based.
